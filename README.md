# Music Tour Revenue Analysis

This project explores and analyzes a dataset of the highest-grossing music tours.

The goal of this analysis is to understand patterns in concert tour revenue, artist performance, and trends across years using Python and Pandas.

## Dataset Overview

The dataset contains information about top music tours including:

* Artist
* Tour title
* Actual gross revenue
* Adjusted gross revenue
* Number of shows
* Average gross per show
* Year(s) of the tour

## Data Processing Steps

1. Data Exploration

   * Checked dataset structure, columns, and statistics.

2. Data Cleaning

   * Removed unnecessary columns.
   * Converted revenue columns from string format to numeric.

3. Data Analysis

   * Identified artists with the highest revenue.
   * Found artists appearing most frequently in the dataset.
   * Analyzed revenue trends by year.
   * Determined tours with the highest average gross.

## Key Insights

* Taylor Swift appears among the highest-grossing tours.
* Some artists dominate the list with multiple tours.
* Tour revenue varies significantly depending on the number of shows and average gross.
